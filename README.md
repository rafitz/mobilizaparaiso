# 🚌 Mobiliza Paraíso

Sistema web para visualização da localização de ônibus em tempo real, com rota no mapa e simulação de movimento.

---

## 📌 Sobre o projeto

O **Mobiliza Paraíso** é uma aplicação que simula o rastreamento de ônibus, permitindo que o usuário visualize:

* Localização do ônibus no mapa
* Rota percorrida
* Tempo estimado e atualização em tempo real
* Interface semelhante a um aplicativo mobile

---

## 🚀 Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Node.js
* Leaflet (mapa interativo)

---

## ▶️ Como executar o projeto

### 🔹 1. Clonar o repositório

```bash
git clone https://github.com/nunesbi/mobilizaparaiso.git
cd mobilizaparaiso
```

---

### 🔹 2. Instalar dependências do backend

```bash
npm install
```

---

### 🔹 3. Rodar o servidor

```bash
node server.js
```

O servidor ficará disponível em:

```
http://localhost:3000/location
```

---

### 🔹 4. Abrir o frontend

Abra o arquivo:

```
index.html
```

Recomendado usar a extensão **Live Server** no VS Code.

---

### 🔹 5. Utilizar o sistema

* Clique no botão inicial
* Visualize o mapa
* Acompanhe o movimento do ônibus na rota

---

## 🧠 Funcionalidades

* Mapa interativo
